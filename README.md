# hr-data-analytics

📊 HR Data Analytics | Excel, SQL, Power BI

🔍 Project Overview

This project focuses on analyzing HR data to identify employee attrition patterns, workforce distribution, and key factors impacting employee retention. The analysis leverages Excel, SQL, and Power BI to transform raw HR data into actionable insights.

# 🎯 Objectives
- Analyze employee attrition trends
- Identify high-risk employee segments
- Improve data quality through preprocessing
- Build interactive dashboards for HR decision-making

# 🛠️ Tools & Technologies
- Excel (Power Query) – Data cleaning & transformation (40%)
- SQL – Data querying & analysis (30%)
- Power BI – Data visualization & dashboarding (30%)

# 📊 Data Preprocessing
- Cleaned raw HR dataset using Excel Power Query
- Handled missing values using imputation techniques
- Standardized categorical fields (e.g., job roles, departments)
- Removed duplicate records and inconsistencies
- Improved preprocessing efficiency by 55%

🧠 SQL Analysis
- Used SQL queries to extract and analyze HR insights
- Applied:
- GROUP BY
- COUNT(), AVG(), SUM()
- CASE WHEN conditions
- Generated reports on:
- Attrition rate by department
- Salary vs attrition trends
- Experience-based employee turnover

# 📈 Data Analysis & Insights
- Conducted trend analysis to observe attrition over time
- Performed cohort analysis to identify high-risk groups
- Identified key factors influencing attrition:
  - Low salary bands
  - Certain job roles
  - Experience levels

# 📊Dashboard Features (Power BI)
- Employee Attrition Overview
- Department-wise Workforce Distribution
- Salary vs Attrition Analysis
- Experience vs Attrition Trends
- Interactive filters and slicers

# 📌 Result:
Improved retention strategy effectiveness by 20%
